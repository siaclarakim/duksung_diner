❤️**덕성여대 맛집 추천 웹사이트**  
덕성여대 주변의 맛집을 소개하는 웹사이트입니다.  
인터넷 기초 과목의 과제로 제작되었으며, **HTML**과 **CSS**를 이용하여 구성하였습니다.  
**주요기능**  
-덕성여대 주변 추천 맛집 10곳 리스트  
-각 음식점의 상세 정보 (이미지, 위치, 평점, 구분)  
-다이닝 코드 링크와 연결  
-페이지 내 목차 이동 앵커 기능  
-시맨틱 태그의 활용  
**사용 기술**  
-HTML5  
-CSS3  
-GitHub Pages
